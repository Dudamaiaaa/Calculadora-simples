   ## 💡 Sobre o projeto

   Calculadora simples com as 4 operações básicas (soma, subtração,
   multiplicação e divisão). Projeto 100% front-end (HTML, CSS e
   JavaScript puro), sem backend.

   ## 🚀 Como usar

   Abra o arquivo `index.html` em qualquer navegador.
